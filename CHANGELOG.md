# Changelog

## [1.4] - 2025-06-29

### 新增

- `formatDuration()` 工具函数（注入脚本 + popup）
- 历史记录新增 `turns`（持续轮数）和 `duration`（持续时长，ms）字段
- 当前模型卡片：模型名、轮数+时长、开始时间
- 发消息时自动刷新下拉菜单数据（如已展开）
- 使用限制信息新增「推测：额度是逐条恢复，不是到点重置」

### 改进

- **卡片布局**（popup + dropdown 统一）：
  - 头部仅保留 `📊 模型监控` + `ⓘ`
  - 模型卡片：🟢/🔴 + 模型名、`X轮 \| XhXm`、开始时间
  - 近3h / 近24h 统计嵌入卡片右侧
  - 历史独立区域，标题"模型切换历史"
  - 历史记录三行格式（模型名、轮数\|时长、时间戳）
  - 当前模型从历史列表排除
- **清除统一**：单按钮「🗑️ 清除监控数据」— 清空消息统计、历史、模型状态

- **持续时间**：`<1分钟` 取代 `刚刚`
- **使用限制文案**：`约10条/5小时`、`160条/3小时`、`Plus 手动选;Go 10条/5小时`

### 移除

- 独立的「清零消息计数」按钮（popup + dropdown）
- 状态行（状态点 + 已降智/正常 + 切换次数）
- 「切换 X 次」文本
- 近24h 旁的「(参考)」
- 无用 CSS 类
